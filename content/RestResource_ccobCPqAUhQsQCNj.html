User Records Filtered by IC
  • URL: https://<host>/api/vds/users/ic/{IC_ACRONYM}
  • Request Type: GET
  • Result Format: JSON or XML formatted array of user objects containing the users' NED and AD information (refer to VDS_InternalView_SchemaProd_20180831.xlsx for list of exposed fields).
  • Default response format: XML
  • Request headers: Authorization (for basic authentication), Accept (for requested response format. optional - defaults to application/xml)
  • This service makes an LDAP call to VDS to get all users falling under the specified IC.
