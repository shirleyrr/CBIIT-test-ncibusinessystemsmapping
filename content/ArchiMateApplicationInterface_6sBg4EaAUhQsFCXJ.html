 UserInfo RestAPI : Class Diagram

User Info API consists of a set of REST web services which pull identity information from the Virtual Directory Server (VDS) and NED. VDS provides an integrated view of NED and Active Directory (AD) identity information for users. Additionally, NED tracks historical change information (of NED information only, AD information changes are not tracked) and exposes an API which returns user account changes within a given time range. Further integration of property information hosted in nVision/Sunflower and additional user information from NCI Frederick databases is currently under development.
