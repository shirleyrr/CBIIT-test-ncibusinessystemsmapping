Description

 

This API returns records that have changed in NED within a given time range. Note that there is no change tracking feature for AD information. Therefore, periodic full updates from VDS are necessary in order to avoid getting out of sync with AD changes.

  • Request Body format: JSON or XML formatted object containing the Icore Site and a time range from which changed records are requested.
  • Default Response Format: XML
  • Resuest headers: Authorization (for basic authentication), Accept (for requested response format. optional - defaults to application/xml)
